**Avatar** is an initials circle in parent colour coding — Copara never uses photos. Use it for people in family lists, request rows, and the account header.

```jsx
<Avatar name="Lisa Wagner" tone="wald" size={40} />
<Avatar name="Markus Berg" tone="apricot" />
<Avatar name="Tom Baumann" tone="schiefer" />
<Avatar name="Oma Anna" tone="neutral" />
```

- `tone`: `wald` = parent A (you), `apricot` = parent B, `schiefer` = parent C (patchwork / third co-parent), `neutral` = grandparents & helpers.
- Colour coding is symmetric and consistent across calendar cells, avatars and stats — keep a person's tone the same everywhere.
